# Download Page

Github is often difficult to navigate for people that aren't used to using it. Even reaching the download page for software they want can be hard! Add this repository to a gh-pages branch in your repository, update the information in `/src/content`, set up the github pages, and your application will have a much more easy-to-access download page! Further customization can be achieved in `/src` if desired.
